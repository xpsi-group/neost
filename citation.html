When using NEoST for EOS inference using the chiral EFT + PP/CS extension, please also cite either Raaijmakers et al. (2021) or Greif, Raaijmakers et al. (2019).

If you wish to cite published appliations of the NEoST package you can find a list of papers that have used the software via Publications.

NEoST also has numerous dependencies that users also benefit from, which should ideally be acknowledged as well.
